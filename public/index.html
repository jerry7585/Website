Website Portfolio

Web Design, Frontend, HTML, CSS, Javascript

You are experiencing this project right now! This is my first web dev expereince, being able to create a website from nothing is a great way to showcase my mastery. I've used HTML/CSS/Javascript for the foundation of the website, then I included many libraries and frameworks like wowjs animations, jQuery, bootstrap, font-awesome, and more to showcase my ability and other projects.

DataVision

Backend, Java, API, Android Studio

An app developed in Java to leverage the data and conclusions gathered from Newark's open data to provide a prediction of how long a user has to wait for a service to be completed. This app takes into account social, cultural, area of issue, etc to give a reliable estimation.

Newark Data Parser

Backend, Java, API

A backend focused project which leverages Java's ability to retreive data online through excel sheets and API. Being able to parse through millions of lines of data very quickly to allow the user to draw conclusions that can be used for other programs.

Library Account Manager

Backend, C++

One of my first big projects that was submitted as a final project for a class. It simulates an account keeper for students to register and input the textbooks they are borrowing for the semester. Included features such as admin, late fees, check out, check in, create/delete account, change password. Fun fact: It was actually used by the semester after mine to keep track of who borrowed the textbooks from the professor!
